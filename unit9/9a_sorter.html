String Sorting



PLEASE READ CAREFULLY!!

Enter the string to be sorted in the texbox shown below. Each word or number must be seperated by a space. For example
123 234 -345 1234567 -9987         -is a valid entry.
Similarly
Hello world this is a test string         -is also a valid entry.

Once completed press the "submit" button . You will be alerted and your submission will be display.You can then sort the string alphabetically or numerically by clicking the Sort buttons provided below the text box.
